# NorthFace Banchee 登山包测评

# 一份不专业的测评报告

# 型号

北面 Banchee50 XS/S

# 颜值

银灰 + 深蓝的组合太帅气了，兼具海王小可爱的风采，如果不装满的话会显得不挺拔，装满应该是满分颜值。

# 设计

面料（210D Ripstop Weave Nylon**）：用水杯淋了一下是防水的，小雨应该可不带防雨罩，有反光的部分，但是很吸灰尘。

Dyno Lite System：合理的功能分区，确实设计了超过 8 个口袋（头包，可以挂上或者敞开口的侧兜，以及有两个 Z 字型交叉的大口袋等等），在肩带上有两个 tool holders 可以顺手放小东西。

OPTIFIT™ harness system：大概是给所有的带子都配置了一个它应该在的一个或者几个很小的线圈的约束，我简直看呆了，设计师非常严谨。

重心调节是单端调节，两端自动平衡（Banchee 系列的特点），背上去手拉带子就可以调，很方便，卡扣有一道保险防滑，貌似就是你忘了按卡扣带子也不会缩回去。

背负：背负系统是 T6 铝框 + 透气的背板，自重 1520 克，对女生友好（背板较硬且贴合腰部分的曲线，腰带比较粗，能受力均匀的包裹你的胯部），背起来很舒服。最高负重：20kg。

# 亮点

1. 背负系统是外围铝框（此框中间还有横杠加强）加中间竖铝条，两块 PE 板支撑，主背负一块较厚大的 PE 板，背板泡沫那还块小薄的 PE 板整型（背板的透气设计有个专利名，看起来也不错）。背负支撑的坚固性在目前轻量化大潮下显得难能可贵。

2. 重心调节是单端调节，两端自动平衡，也就是广告上说的那个在申请的专利。这个应该是从它家可调节的背负系统发展来的，之前在 Banchee 系列里有看到。这两个调节系统蛮有意思。重心带的卡扣是双重的，有一道保险防滑，但背负调节只有一道卡扣，不知道用久了后，松动是否会造成背负滑动。

3. 主仓一个大 J 型拉练可以全部打开，而且一个亮点是，老款重装包一般都会有底仓拉链，这个 J 型拉练把主仓侧开和底仓拉链全并设计成一条了，共三个拉链头，也蛮方便的。之前我一直想入手 lowe alpine 家的 alpmayo 就是因为比 CT 系列少了这条底仓拉链省了重。

4. 可拆的袋鼠仓，拆了可单独做登顶包。相当于一个附包，我称了下这个包就有 211.5g. 要轻量化点大可以拆掉。因为本身它还有一个稍小点的弹力面料袋鼠仓。此亮点适合你有一个体弱但从不放心打定主意跟着你爬山的 GF。

5. 可调节腰带，这包的腰带主骨架是一块硬塑料板，上面穿插上厚实的腰带（可调节位置），相对于其它重装包的腰带，它只有包裹的位置是厚的，省了些重量，但支撑耐久就需要长久考验了。此腰带也是可上下转动的，转动方式和范围和我之前老款 Gregory B70 非常相似（新款 B65 B75 我没背过），个人更喜欢这种方式，不太喜欢 Thule Guidepost 那种。后者这种像个滚轴一样，重心稍偏包就歪了。鸟家的活动腰带只试背过，不好比较（主要是穷）。

# 缺点

1. 用料相对较次，主要是大身面料，虽然标的是 210D 及 420D 的尼龙，手感却不像我所有其它包的尼龙一样，较硬，有一种绦纶的感觉，不看牌子只摸它肯定会得出这就一山寨厂家劣包的结论，这面料特粘灰尘，不知道是否静电原因，有吸尘器的功能。另外，工具带之类相当简化，那绑绳用料好次（这个能用就行，可接受）

2. 空间较小，重量虚标，我看的是 Griffin 65, 尺寸空间比我的 Stein 62 还小。s/m 码标重 2268g, 实称 2380g, 也就是重量实际达到了 Griffin 75 l/xl 码数据。

3. 水袋仓不够大，对于我来说不是缺点，我从来不用带饮水管的水袋，只用储水袋。

By 胡椒 2022 春 激动的写于图书馆
